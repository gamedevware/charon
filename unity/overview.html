Unity Plugin Overview

The Unity plugin version is a plugin for the Unity game engine that allows users to design and model game data, as well as generate source code for it, directly within the Unity editor. The Unity plugin version is used by game developers who are already using the Unity engine for their game development, as it allows them to seamlessly integrate game data design and modeling into their existing workflow.
