Get Charon Version
==================

Gets version of ``Charon.exe`` application.

---------------
 Command
---------------

.. code-block:: bash

  # Windows
  Charon.exe VERSION
  > 2023.2.3-alpha
  
  # Linux or OSX
  mono Charon.exe VERSION
  > 2023.2.3-alpha
  
---------------
 Parameters
---------------

`Universal parameters <universal_parameters.rst>`_ apply to this command.
